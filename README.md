# Table Extractor Chrome Extension

Chrome MV3 浏览器插件，用于：

- 直接捕捉页面数据: 将页面中类似表格的内容导出为 `.xlsx` 文件，或者 Markdown 格式。

- 捕捉分页数据链接: 以变量方式下载 `.xlsx` 文件。

## 直接捕捉页面数据支持的结构

- 原生 `<table>` 表格。
- `<ul>` 列表表格：每个 `<li>` 作为一行，`li` 下的直接子元素作为列；如果 `li` 只有一个直接子元素，则会继续读取该子元素下的直接子元素作为列。

## 捕捉分页数据链接

- 通过捕捉两次点击下一页/上一页或类似按钮的方式获取链接，并填写变量；在页面中自动下载数据，导出目标格式数据表。

## 扩展加载方式

1. 打开 Chrome 扩展管理页：`chrome://extensions/`。
2. 开启「开发人员模式」。
3. 点击「加载解压缩的扩展」按钮，选择本项目目录。
4. 在扩展按钮下, 可见 `Table Extrator` 扩展

## 扩展的使用

扩展有两个功能: 捕捉页面数据和捕捉分页数据

### 捕捉页面数据

1. 打开包含表格的页面，
2. 点击扩展 - `Table Extrator`。
3. 默认是 `扫描表格` 页签，点击「扫描表格」。
4. 在扫描结果列表中勾选要导出的 table 或 ul 表格；鼠标移动到列表项时，页面会高亮对应元素。
5. 可使用「全选」「去选」快速操作，最后点击「导出 XLSX」。

导出文件会通过浏览器下载为 `.xlsx`。

### 捕捉分页数据链接

1. 打开包含分页数据的页面，通常这类页面有“上一页”，“下一页”等按钮或链接
2. 点击扩展 - `Table Extrator`。
3. 默认是 `扫描表格` 页签，选择 `网络分页` 页签。
4. 点击开始监听;
5. 在页面中点击下一页两次，注意等待数据刷新出来。注意，此步操作扩展菜单会消失是正常现象。
6. 按照 步骤 2 的操作, 再次点击扩展 - `Table Extrator`;
7. 此时的默认页签是 `网络分页`, 可以看见链接和参数名等信息, 如果不可见, 可以使用刷新捕获的方式获取;
8. 调整变量起始值, 结束值, 步长参数, 点击 `导出网络分页XLSX` 按钮;

扩展会在导出数据后将数据下载为 `.xlsx` 文件。

该功能适用于返回 JSON 数组数据的分页接口。若接口使用加密参数、强签名、WebSocket 或非 JSON 响应，可能无法自动批量提取。
