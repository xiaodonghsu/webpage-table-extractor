# Table Extract Chrome Extension

一个 Chrome MV3 浏览器插件，用于在网页上选择表格并导出为 `.xlsx` 文件。

## 支持的结构

- 原生 `<table>` 表格。
- `<ul>` 列表表格：每个 `<li>` 作为一行，`li` 下的直接子元素作为列；如果 `li` 只有一个直接子元素，则会继续读取该子元素下的直接子元素作为列。

## 使用方式

1. 打开 Chrome 扩展管理页：`chrome://extensions/`。
2. 开启「开发者模式」。
3. 点击「加载已解压的扩展程序」，选择本项目目录。
4. 打开包含表格的页面，点击扩展图标。
5. 点击「扫描表格」。
6. 在扫描结果列表中勾选要导出的 table 或 ul 表格；鼠标移动到列表项时，页面会高亮对应元素。
7. 可使用「全选」「去选」快速操作，最后点击「导出 XLSX」。

导出文件会通过浏览器下载为 `.xlsx`。

## 网络分页提取

对于分页表格，可以使用 popup 下方的「网络分页提取」：

1. 点击「开始监听」。
2. 回到页面点击一次分页按钮，例如「下一页」。
3. 回到插件 popup，点击「刷新捕获」。
4. 选择捕获到的 JSON 请求，填写页码参数、起始页和结束页。
5. 点击「导出网络分页 XLSX」。

该功能适用于返回 JSON 数组数据的分页接口。若接口使用加密参数、强签名、WebSocket 或非 JSON 响应，可能无法自动批量提取。
